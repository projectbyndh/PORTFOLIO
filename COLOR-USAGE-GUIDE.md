## ‚úÖ Best Practices

### When to use `#0D1641` (Brand Dark):
- ‚úÖ Primary text on light backgrounds
- ‚úÖ Button backgrounds for CTAs
- ‚úÖ Strong emphasis headings
- ‚úÖ Navigation text (inactive state)

### When to use `#26a8df` (Brand Primary):
- ‚úÖ Accent colors and highlights
- ‚úÖ Interactive elements (hover states)
- ‚úÖ Icons and decorative elements
- ‚úÖ Active navigation states
- ‚úÖ Loading indicators
- ‚úÖ Button backgrounds (primary actions)

### When to use Gradient Classes:
- ‚úÖ Hero section main headings
- ‚úÖ Section title highlights
- ‚úÖ Feature names that need extra emphasis

## üîß Quick Reference

**CSS Variables** (App.css):
```css
--brand-dark: #0D1641
--brand-dark-rgb: 13, 22, 65

--brand-primary: #26a8df
--brand-primary-rgb: 38, 168, 223

--gradient-brand: linear-gradient(to right, #26a8df 0%, #26a8df 50%, #2D5F8C 100%)
--gradient-brand-alt: linear-gradient(to right, #26a8df 0%, #2D5A7B 50%, #26a8df 100%)
```

**Utility Classes**:
- `.text-gradient-brand` - Main brand gradient
- `.text-gradient-brand-alt` - Alternative gradient
- `.bg-brand-dark` - Dark background
- `.bg-brand-primary` - Primary background
- `.text-brand-primary` - Primary text color
